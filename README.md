# 🚀 AI-Based Resume Skill Gap Analyzer  
An intelligent AI-powered platform designed to analyze resumes, identify skill gaps, and provide personalized learning recommendations for career growth. The system helps students and professionals understand their current skills, compare them with industry requirements, and receive adaptive suggestions to improve employability and technical expertise.
Built as a collaborative team project, this application combines AI-driven analysis with modern full-stack development to create an interactive and user-friendly experience.  

🌐 Live Demo: https://ai-onboarding-engine.vercel.app/
---

# 📌 Overview  
The AI-Based Resume Skill Gap Analyzer helps users:  
- Analyze resumes intelligently  
- Extract technical and professional skills  
- Identify missing industry-required skills  
- Receive personalized learning recommendations  
- Improve career readiness and technical growth  
The project uses AI-driven logic and structured datasets to bridge the gap between existing skills and industry expectations.  

---

# ✨ Features  
## 📄 Resume Skill Extraction  
Extracts relevant skills and technologies from uploaded or entered resume data.  

## 📊 Skill Gap Analysis

Provides adaptive learning suggestions and career improvement pathways.

# 🚀 How to Run Locally  

## 1️⃣ Clone the Repository  

```bash
git clone https://github.com/srishtimishra30/AI_Based_Resume_Skill_Gap_Analyzer.git

cd AI-Based-Resume-Skill-Gap-Analyzer
```

---

## 2️⃣ Backend Setup  

```bash
cd backend

pip install -r requirements.txt

uvicorn backend.app:app --host 0.0.0.0 --port 8000 --reload
```

Backend runs on:  

```bash
http://127.0.0.1:8000
```

---

## 3️⃣ Frontend Setup  

Open a new terminal:  

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:  

```bash
http://localhost:5173
```
